## **Project Purpose**

LvLUp Gaming is a React-based web application that showcases top-rated indie and AAA games. Users can browse games, view details, and log in with their Google account for a personalized experience. The project demonstrates integration with Firebase Authentication, React Router, and modern frontend development practices using Vite and Tailwind CSS.

---

## **Key Features**

- **Google OAuth Login:** Secure authentication using Firebase. Users can log in with their Google account.  
- **Dynamic Game Listing:** Browse a collection of games with images, ratings, and descriptions.  
- **Responsive Design:** Fully responsive UI for desktop, tablet, and mobile.  
- **Interactive UI:** Smooth animations using React Spring for text and UI elements.  
- **Client-Side Routing:** Multi-page navigation handled with React Router.  
- **Profile Display:** Logged-in users can see their Google profile name and photo.  
- **Fast Deployment:** Hosted on Netlify for instant live access.

---

## **Tech Stack & NPM Packages**

- **Frontend:** React, Vite, Tailwind CSS  
- **Authentication:** Firebase Authentication (Google Sign-In)  
- **Routing:** React Router DOM  
- **Animations:** React Spring  
- **Deployment:** Netlify
